# 🌐 My WebSite

這是一個自用小型網站，整合了日常使用的實用工具與練習遊戲，採用 GNU General Public License v3.0 許可證授權。

## 🏠 首頁
乾淨直觀的入口頁面，提供快速導航至各功能模塊：
- 支持深色/淺色模式即時切換
- 自動檢測屏幕尺寸，對寬度小於767px或高度小於500px的設備進行訪問限制（避免體驗不佳）
- 完全響應式設計，兼容主流桌面設備及平板設備


## 📘 作業查詢工具（My HW List）
輕量級作業管理工具，幫助高效追蹤學習任務：
- 清晰展示作業列表，按科目分類呈現，包含ID、科目、名稱、發出日期、到期日期、組別、狀態及備註等詳細信息
- 支持多維度篩選（日期、科目、到期狀態）、數據統計（今日家課數、今天到期數等）
- 自動適應深色/淺色模式，並通過ARIA屬性強化可訪問性
- 數據加載失敗時支持手動上傳JSON文件，確保使用靈活性
- 通過GitHub Actions自動化工作流每日更新數據（香港時間17:00），支持手動觸發強制更新
- 支持將家課表輸出為圖片格式


## 🎮 百分比變化練習遊戲（P-Change Game）
專注於數學百分比變化計算的互動練習工具：
- 隨機生成多種題型（求新值、求原值、計算變化百分比等）
- 強化百分比增減概念的理解與應用
- 適合快速練習，提升計算熟練度
- 支持中英文語言即時切換，適應不同使用場景
- 🎉 隱藏小彩蛋，需通過特定操作解鎖
- 採用高DPI適配的Canvas繪圖，保證顯示清晰度
