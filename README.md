# 🌐 My WebSite

這是一個自用小型網站，有日常使用的實用工具與練習遊戲。


## 🏠 首頁
乾淨直觀的入口頁面，提供快速導航，支持深色/淺色模式切換，適配各種設備屏幕。

### 📘 作業查詢工具（My HW List）
一個輕量級的作業管理工具，幫助高效追蹤學習任務：
- 清晰展示作業列表，分類呈現各科目任務
- 支持關鍵字搜尋、條件篩選與數據統計
- 自動適應深色/淺色模式
- 介面簡潔直觀
- 透過自動化工作流定期更新數據，確保內容即時性


### 🎮 百分比變化練習遊戲（P-Change Game）
專注於數學百分比變化計算的互動練習工具：
- 隨機生成多種題型（求新值、求原值、計算變化百分比等）
- 強化百分比增減概念的理解與應用
- 適合快速練習，提升計算熟練度
-  🎉隱藏小彩蛋，需要**特定條件**解鎖
- 支持中英文語言切換，適合不同使用場景
