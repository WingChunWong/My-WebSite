# 🌐 My WebSite

這是一個我個人建立的小型網站，有一些我平常會使用的小工具、小遊戲。
網站的目標是簡單、方便，也讓我能夠自由地嘗試各種想法。

## 🏠 首頁

乾淨的入口頁面，可以快速前往其他遊戲與工具。

### 🎮 P-Change Game（百分數變化練習遊戲）

一款專門用來練習「百分比變化」概念的小遊戲：

* 隨機題目的百分比變化練習
* 增加／減少百分比概念加強
* 適合快速做幾題提升手感
* **內含小彩蛋 🎉，需要特點條件觸發！**

### 📘 作業查詢工具（My HW List）

一個能快速查看作業資訊的輕量小工具：

* 清楚的作業列表
* 搜尋、篩選與統計
* 支援深／淺色模式
* 介面簡潔、使用方便
* 網站會自動更新資料，保持內容最新
